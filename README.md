# Network Intrusion Detection System using Snort

## Project Overview

This project was developed as part of the **CodeAlpha Cyber Security Internship**. The main objective of this project is to implement a **Network Intrusion Detection System (NIDS)** using **Snort** in **Kali Linux** to monitor network traffic and detect suspicious activities.

The project demonstrates how Snort can analyze network packets in real time and generate alerts when suspicious traffic is detected using custom rules.

---

## Objective

The main objectives of this project are:

- To understand the fundamentals of Network Intrusion Detection Systems (NIDS)
- To install and configure Snort in Kali Linux
- To monitor network traffic in real time
- To create custom detection rules
- To identify suspicious activities in a network environment
- To improve practical knowledge of network security and threat detection

---

## Tools and Technologies Used

- **Snort**
- **Kali Linux Operating System**
- **Terminal / Command Line Interface**
- **Network Monitoring**
- **Custom Snort Rules**
