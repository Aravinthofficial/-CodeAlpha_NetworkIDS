# Network Intrusion Detection System using Snort

## Project Overview
This project was developed as part of the CodeAlpha Cyber Security Internship. The objective of this project is to implement a Network Intrusion Detection System (NIDS) using Snort to monitor network traffic and detect suspicious activities.

## Objective
- Understand the basics of Intrusion Detection Systems (IDS)
- Install and configure Snort
- Monitor network traffic in real time
- Create custom alert rules
- Detect suspicious activities

## Tools and Technologies Used
- Snort
- ubuntu Linux
- Terminal / Command Line Interface
- Network Monitoring
